# Python Calculator

Простой калькулятор на Python с базовыми арифметическими операциями.

## Функции

- `add(a, b)` - сложение
- `subtract(a, b)` - вычитание  
- `multiply(a, b)` - умножение
- `divide(a, b)` - деление

## Использование

```python
from calculator import add, subtract

result = add(10, 5)
print(result)  # 15
